מארז Benefit

מה יש במארז הזה?

כל מה שצריך כדי להעמיד את סוכן ה-Benefit שלכם: הסוכן הבנוי להדבקה, קובץ course-structure שהוא המפה של מחברת ה-Notion שלכם, וארבעה סקילים ארוזים כקובץ ZIP מוכן להעלאה. מורידים מכאן ומתקינים. אם משהו לא עובד, נסו שוב, נסדר במפגש.

איך מרכיבים את Benefit

ארבעה צעדים, בסדר הזה. הכל להורדה מהעמוד הזה.

1
הורידו את הסוכן המוכן והדביקו אותו להוראות הפרויקט (Project instructions) בקלוד.
2
הורידו את קובץ מבנה-הקורס והעלו אותו לפרויקט כקובץ Knowledge, יחד עם שאר חומרי-היחידות.
3
הורידו והתקינו את ארבעת הסקילים לפי ההסבר שבתחתית העמוד.
4
חברו את Notion דרך החיבור (MCP), עם הרשאת קריאה וכתיבה, ואשרו את כלי הכתיבה כשהוא מבקש.
2

קבצי הידע

הסוכן הוא system prompt מלא שנכנס להוראות הפרויקט (אפשר להעתיק או להוריד), ומבנה-הקורס עולה כקובץ Knowledge.

הסוכן המוכן

knowledge/benefit-agent.md

סוכן Benefit בנוי ומוכן, system prompt מלא. הדביקו את התוכן להוראות הפרויקט. אתם לא בונים אותו מאפס.

הורד

מבנה הקורס

knowledge/course-structure.md

המפה של מחברת ה-Notion שלכם וכללי-התיוק. זה קובץ הידע שהופך את התיוק לאוטומטי. העלו אותו לפרויקט כקובץ Knowledge.

הורד את המבנה
3

ארבעת הסקילים

כל סקיל מורד כקובץ ZIP מוכן להעלאה לקלוד. שניים מתקינים ומריצים בכיתה (שמירה ושליפה, שני הכיוונים של המאגר), ושניים בבית (ייעוץ STUDIO ובניית פרומפט MICRO). אפשר גם להעתיק את הטקסט אם מעדיפים את ה-Skill Creator.

בכיתה

שמירה ויצירת עמוד

benefit-save-page

מתייק תוכן ב-Notion במקום הנכון לפי כללי-התיוק, ויוצר או מעדכן עמוד. זה הסקיל שכפתור "העבר ל-Notion" מפעיל.

הורד ZIP

benefit-retrieve

הורד ZIP

איך מתקינים סקיל בקלוד

כל סקיל יורד כקובץ ZIP מוכן. ה-ZIP כבר בנוי נכון, אין צורך לדחוס דבר.

  1. ודאו שהרצת קוד (Code execution) מופעלת בהגדרות, תחת Capabilities או Features.
  2. הורידו את קובץ ה-ZIP של הסקיל מהכרטיס למעלה.
  3. בקלוד, היכנסו ל-Settings, ובמקטע Skills בחרו העלאה (Upload skill).
  4. העלו את קובץ ה-ZIP. חזרו על זה לכל אחד מארבעת הסקילים.
  5. פתחו את פרויקט Benefit, והסקילים זמינים לשימוש.
חלופה: אפשר ליצור סקיל חדש דרך ה-Skill Creator בקלוד ולהדביק לתוכו את הטקסט מכפתור "העתק טקסט", בלי ZIP. אם ההתקנה לא עובדת, נסו שוב, נסדר במפגש.
