# שמירה ויצירת עמוד

## מתי להפעיל
זה הסקיל שכפתור "העבר ל-Notion" בפלטפורמה מפעיל. הפעל אותו כשמגיע תוכן עם הכותרת "Benefit, הפעל את סקיל השמירה", או כשהתלמיד מבקש "העבר ל-Notion", "תכניס לתובנות שלי", או "תוסיף לדברים שגיליתי".

## מה הסקיל עושה
מתייק את התוכן במקום הנכון במחברת ה-Notion לפי כללי-התיוק, ויוצר עמוד חדש או מעדכן עמוד קיים דרך Notion MCP. הכתיבה תמיד באישור התלמיד.

## קלט ופלט
קלט: תוכן ויעד. פלט: עמוד מתויק או מעודכן ב-Notion.

## הפרומפט שהכפתור שולח
הפלטפורמה מעתיקה את תוכן היחידה ומעליו את ההנחיה הבאה. את `{שם היחידה}`, `{N}` ו-`{תחנה}` היא ממלאת בעצמה לפי הסיווג של היחידה:
```
זה התוכן של היחידה «{שם היחידה}» (שיעור {N}, STUDIO: {תחנה}).
Benefit, הפעל את סקיל השמירה: תייק אותו תחת «שיעור {N}» ← «{שם היחידה}»,
עם שורת STUDIO: {תחנה}. אם העמוד קיים — עדכן; אם לא — צור. אל תיגע בעמודים אחרים.
```
זהה את הערכים מתוך ההנחיה ופעל לפיהם בדיוק. אל תיגע בעמודים אחרים.

## כללי-התיוק
1. **תוכן יחידה** (כפתור "העבר ל-Notion"): תייק תחת `שיעור N ← <יחידה>`, עם שורת `STUDIO: <תחנה>` בראש העמוד. אם העמוד קיים, עדכן. אם אינו קיים, צור.
2. **"תכניס לתובנות שלי":** הוסף לעמוד `התובנות שלי`. בלי תיוג STUDIO.
3. **"תוסיף לדברים שגיליתי" או "תוסיף לגילויים שלי"** (משהו מהאינטרנט): הוסף לעמוד `דברים שגיליתי בעצמי`. בלי תיוג STUDIO.
4. **נושא חדש בלי בית קיים:** צור עמוד חדש תחת השיעור המתאים. אם הנושא חוצה-שיעור, צור תחת עמוד-האב Benefits. תייג STUDIO אם רלוונטי.

המפה המלאה של המחברת נמצאת בקובץ מבנה-הקורס. הסתמך עליו לכל איתור של עמוד-אב, שיעור או יחידה.

## איך לעבוד
1. זהה את היעד: יחידה, תובנה אישית, גילוי חיצוני, או נושא חדש.
2. אתר לפי-צורך אם עמוד היעד כבר קיים. אל תטען את כל המחברת.
3. הכן את התוכן. בעמוד-יחידה ודא ששורת `STUDIO: <תחנה>` נמצאת בראש העמוד.
4. הסבר לתלמיד מה אתה עומד לכתוב ולאן, וחכה לאישור.
5. אחרי אישור, כתוב דרך Notion MCP: צור עמוד חדש או עדכן את הקיים.
6. אשר לתלמיד מה נשמר ואיפה.

## אם משהו לא עובד
אם פעולת הכתיבה נכשלת, ודא שחיבור ה-Notion פעיל עם הרשאת כתיבה ושכלי הכתיבה אושר. אם זה עדיין לא עובר, אמור זאת בבירור: נסו שוב, נסדר במפגש.
